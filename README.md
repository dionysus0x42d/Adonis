# Adonis

阿多尼斯是希臘神話中的美男子。

這是一個基於 GVDB 的影片資料庫。

---

## 🌐 立即使用

**👉 [點擊這裡訪問查詢網站](https://dionysus0x42d.github.io/Adonis/)**

---

## ✨ 有什麼功能？

### 🎬 演員查詢
- 搜尋演員名稱
- 按公司篩選
- 看演員的作品數量和角色統計
- 查看演員在各公司的藝名

### 🎥 作品查詢
- 搜尋作品代碼或標題
- 按公司、演員、日期篩選
- 按喜歡的方式排序
- 檢視作品的詳細資訊和標籤
- 展開專輯查看所有片段
