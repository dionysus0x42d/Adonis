# GVDB - 同志影視資料庫

一個簡單好用的成人影視作品查詢系統，讓您輕鬆找到想看的演員和作品。

---

## 🌐 立即使用

**👉 [點擊這裡訪問查詢網站](https://dionysus0x42d.github.io/Adonis/)**

網站完全免費且支援離線使用，首次訪問時會自動下載資料到您的瀏覽器。

---

## ✨ 有什麼功能？

### 🎬 演員查詢
- 搜尋喜愛的演員
- 按公司篩選
- 看演員的作品數量和角色統計
- 查看演員在各公司的藝名

### 🎥 作品查詢
- 搜尋作品代碼或標題
- 按公司、演員、日期篩選
- 按喜歡的方式排序
- 檢視作品的詳細資訊和標籤
- 展開專輯查看所有片段

---
